# Presentación 3 – Proyecto Final (Fase 3)

Este repositorio contiene las interfaces públicas del proyecto de crowdfunding correspondientes a la Fase 3 del Proyecto Final.

### Contenido entregado
- Página principal (index.html)
- Exploración de proyectos (explorar.html)
- Detalle del proyecto (detalle-proyecto.html)
- Registro de usuarios (registro.html)
- Autenticación / Login (login.html)
- Roles de Administrador y Usuario
- CSS y JS organizados en carpetas
- Diseño responsive para móvil, tablet y PC

### Cómo ejecutar
Abrir `index.html` directamente en el navegador,
o servir localmente con cualquier servidor web estático.

### Nota
El contenido es estático porque esta fase solo requiere interfaces públicas.
